# Style Guide

## Colors
- Primary: `#667eea`
- Secondary: `#764ba2`
- Accent: `#FF9839`
- Text: `#1F2937` (gray-800)
- Muted: `#6B7280` (gray-500)

## Typography
- Headings: bold, tight line-height
- Body: readable 16px+, 1.6 line-height
- Buttons: medium/semibold, no all-caps

## Components
- Buttons: rounded-full, clear hover/active/focus states
- Cards: `border`, `shadow`, `rounded-xl`
- Modals: accessible titles, focus trap

## Spacing
- Base: 4px unit (`0.25rem`)
- Common gaps: 8/12/16/24px

## Brand Usage
- Use gradients: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)` for hero/primary CTAs
